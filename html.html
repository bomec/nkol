Embedding a youtube video in HTML

To embed a youtube video to HTML, use the Iframe tag instead of the video tag, then follow these steps;

  1. Copy the video's URL
  2. Use the URL as the source
  3. Modify the URL by changing the "watch" to "embed"
  4. Autoplay and mute elements are added in the URL. At the end of the URL simply add a question mark "?", then addautoplay=1 and & mute=1.
  5. To add controls, simply add controls=1. To remove controls, add controls=0 to the URL
  6. In order for the video to play on repeat, add loop=1
