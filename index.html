Borders

  • CSS border width: The border-width property specifies the width of the four borders.
    • Specific Side Widths: The border-width property can have from one to four values (for the top border, right border, bottom border, and the left border
  • CSS border color:The border-color property is used to set the color of the four borders.
    • The border-color property can have from one to four values (for the top border, right border, bottom border, and the left border).
  • The border-radius property is used to add rounded borders to an element

The border-style/border property can be used to combine all these properties together (style, width, and color). But when you want to individually customize an aspect of the border, then you can use them individually (border-width, border-color). You can also combine all the properties to customize one side with the border-top-style/ border-left-style/ border-bottom-style/ border-right-style

CSS Margins

Margins are used to create space around elements, outside of any defined borders.
CSS has properties for specifying the margin for each side of an element:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

CSS Padding

Padding is used to create space around an element's content, inside of any defined borders. With CSS, you have full control over the padding. There are properties for setting the padding for each side of an element (top, right, bottom, and left). CSS has properties for specifying the padding for each side of an element:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

CSS Height and Width

The height and width properties are used to set the height and width of an element. The height and width properties do not include padding, borders, or margins. It sets the height/width of the area inside the padding, border, and margin of the element.
The height is set in pixels and the width in percentage.

CSS Outline

An outline is a line that is drawn around elements, OUTSIDE the borders, to make the element "stand out". CSS has the following outline properties:

  • Outline-style: The ouline-style has the same properties as border-style.
  • Outline-color: The outline color is same as border color properties.
  • Outline-width: Same as border width.
  • Outline-offset: It creates space between the border and the outline.

Font variant

The font-variant property specifies whether or not a text should be displayed in a small-caps font.

In a small-caps font, all lowercase letters are converted to uppercase letters. However, the converted uppercase letters appears in a smaller font size than the original uppercase letters in the text.
