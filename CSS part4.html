CSS Weblayout

We'll study a bit of media queries in this chapter too. Media queries affect the size of an element when the browser is resized.

Very important note

Always add the "* { box-sizing: border-box;}" property at the top of CSS before even starting to code. It helps to automatically group layouts and table together.

For example: notice that in the layouts below the left, middle and right content align side by side without breaking the flow despite the fact that there is space (padding) between them. This is thanks to the border-box.
