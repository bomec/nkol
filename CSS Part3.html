The dropdown



Some basic rules for a proper dropdown

  • The position: The position of the container should always be "relative" and the position of the dropdown should be "absolute".
  • The display: The display of the container should always be "inline-block" (this automatically reduces the width of the container, so you won't need to manually set the width. Try removing this from the code to see what happens) and the display of the dropdown should be "none". The display of the hover should be "block"
  • Add a box-shadow to the dropdown to make it more realistic.

Adding a dropdown menu in a navigation bar require extra properties:
  1. Start by creating a normal navigation bar with links(home, news etc)

CSS Forms

CSS [attribute="value"] selector

The [attribute="value"] is used to select elements with a specific attribute.
For example: If you want to select "a" or link elements which have the "target:_blank" attribute, you'll write:
a[target=_blank]

We are going to use it for forms this way: input[type=text] or input[type=button]

Styling Input fields

There are several types of inputs we studied in HTML but what style are:
  • Text
  • Button
  • Textarea
    • and others...
We'll use the following to style them:
  • Width (in %)
  • Padding
  • Margin
  • Border and border-radius
  • Box-shadow
  • Text color
  • Box-sizing
  • We'll style the div as well.
Note: It is advisable to use the box-sizing property and the display:block property when styling inputs like text, button etc





Focused and Animated inputs

  • You can style the way an input will look like when it gets focus (clicked on) by adding a ":focus" attribute after the selector in CSS, this way;
    e.g input[type="text"]:focus
  • You can also style the label. see CSS code
  • By default, some browsers will add an outline around the input when it gets focus (clicked on) (try clicking the example above to see what I'm talking about). You can remove this behavior by adding outline: none; to the input.
  • Notice that when you click on the input, its width changes. That's thanks to the :focus attribute. Now, you can add some effects to the transition between one width to another using the transition tag.
    (we'll study transitions in advanced chapters)
